# Gtest-App

G検定（AI・統計・ディープラーニング分野）の学習用に作成した  
**単一選択式・逐次回答型の問題演習アプリ**です。

Streamlit を用いて実装されており、PC・スマートフォン（iPhone / Safari 含む）から利用できます。

---

## 特徴

- 問題を1問ずつ表示し、回答・正誤判定を行う
- 「前へ」「次へ」による問題の行き来が可能
- 同じ問題を何度回答しても **最終回答のみを評価**
- 結果画面では以下を表示
  - 総問題数
  - 正解数 / 不正解数
  - 最終的に間違えた問題と解説
- 「もう一度」ボタンで最初からやり直し可能

---

## 動作仕様（重要）

本アプリは以下の仕様を前提としています。

- 各問題は **必ず「判定する」を押してから次へ進む**
- 「前へ」で戻った場合、再度回答・判定が可能
- 同一問題に複数回誤答しても、
  - 結果表示では **最終的に誤答だった場合のみ1回表示**
- 正解数・不正解数は **最終回答ベースで集計**

※ 未回答のまま次へ進むことはできません。

## 起動方法（ローカル）

### 1. リポジトリを取得

```bash
git clone https://github.com/okk4lt0/Gtest-App.git
cd Gtest-App

2. 依存関係をインストール

pip install -r requirements.txt

3. アプリを起動

streamlit run app.py


⸻

スマートフォン（iPhone / Safari）での利用
	•	本アプリは ブラウザ上で動作します
	•	Streamlit Cloud 等にデプロイした URL に
iPhone の Safari からアクセスするだけで利用可能です
	•	特別なアプリのインストールは不要です
